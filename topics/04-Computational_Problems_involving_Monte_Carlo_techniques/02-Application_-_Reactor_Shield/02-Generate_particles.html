Generate Particles

Outline

  • Ensure initial energy follows the specified distribution.
  • Ensure directions follows the specified distributions.
  • Using a class to simplify storing of data.

Task 2A — Using a class to simplify storing of particle data

I would recommend that you use a class to store the particle data (position, direction, energy, etc)

At a minimum you could just use

1
2
class Particle():
    pass

Then create a new particle using 

1
particle = Particle()

Set its values using

1
particle.energy = 15

and read its values using

1
print(particle.energy)

However, for debugging, it is nice to be able to print all of the data stored in one command, so i recommend you define Particle as

class Particle():
    pass

    def __repr__(self):
        return (f"Particle:"
            f"   position = ({self.x:.3f},{self.y:.3f},{self.z:.3f})"
            f"   mu = {self.mu:.3f} "
            f"   azim = {self.azim:.3f} "
            f"   energy = {self.energy:.5f} ")

Now print(particle) we generate output like (once you have set particle.x, particle.y, particle.z, particle.mu, etc )

1
Particle:   position = (0.000,0.000,0.000)   mu = 0.033    azim = 2.146    energy = 1.12256

Task 2B — Implement function generate()

Implement functions generate which "creates" a particle with required properties:

  • a random initial direction, subject to:
    • Let \(\mu\) be the cosine of the angle between the positive \(x\)-axis and neutron velocity vector, then we assumes that \(\mu\sim {\cal U}(0,1)\). This means that all particle generated will penetrate the shield and that particles are more likely to be injected along the normal to the shielding rather than along the tangent.
    • The azimuthal angle, \(\phi\sim{\cal U}(0, 2\pi)\). So problem is effectively two dimensional. However, it is common in reactor calculation to implement problem in 3D so we will follow that practice here.
  • a random energy, \(E\), where \(E_{\min}=0.001 \le E \le E_{\max}=2.5 \text{MeV}\) with \(1/\sqrt{E}\) distribution.

Task 2C — Verify generated particles have required properties

1

Now to get the energies, you get
