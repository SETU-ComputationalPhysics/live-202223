Run Simulation

Task 4A — Simulate a single particle

Write function simulate_particle(rng, debug=False) that simulates the behaviour of a single particle and returns the history of particle positions and the final state reflected, transmitted or absorbed. Outline of algorithm:

  • Generate random particle using generate.
  • Initialise history array at origin.
  • Loop forever:
    • Update particle position using function update
    • Append new position to history
    • If the particle is now to the left of the shield, it is counted as being reflected then increment the number of reflected particles and exit the loop.
    • If the particle is to the right of the shield, it is counted as being transmitted then increment the number of transmitted particles and exit the loop.
    • If the particle is inside the shield, it has collided. A particle that is collided, then is is either absorbed (end of story) or Scattered with a new random direction and a new (lower) energy.
    • Otherwise, particles are *absorbed with probability 10%.
    • Otherwise, generate new particle direction.
