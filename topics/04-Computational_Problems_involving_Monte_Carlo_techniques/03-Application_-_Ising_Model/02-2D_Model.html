2D Isling

Implementation

The 2D Ising model implementation follows very closely the 1D model implementation. In particular define functions:

  • Initialise 2D model init(n).
  • Visualise the model show(model).
  • Simulate the model simulate(model, title, width, filename).
  • Compute the energy per site, energy(model).
  • Compute the magnetisation per site, magnetization(model).

Estimation of the critical Temperature

The curve appears to be \[ f(T) = A \arctan\big[B\left(T-T_{critical}\right)\big] + C \] with parameters \(A\), \(B\), \(C\), and \(T_{critical}\). However, before fitting it would be a good idea to remove all of the noise in the bottom left quarter of the figure. A np.where call will create a logical filter to do this for you. So you should perform the fit on the filtered data subset.
